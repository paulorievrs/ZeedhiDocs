# Containers

Os containers são components que englobam e constroem sua tela, que veem na propriedade de **template** do JSON. Você pode ver elas nativamente nas pastas:

```
project
│   backend
│   ci_scripts   
└───mobile
│   │   assets
│   └───bower_components
│       │   sortablejs
│       │   tek-libraries
│       └───zeedhi-fronted    
│           │   assets
│           └───templates
│               └───dyman
│                   │   component
│                   └───container
 ```
